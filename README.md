## Getting Started (Local)

1. **Clone the repo** (after you create it on GitHub):

   ```bash
   git clone https://github.com/<your-username>/<your-repo-name>.git
   cd <your-repo-name>
   ```

2. **Open locally**
   - Option A: Just open `index.html` in your browser.
   - Option B (recommended): Use a local static server for smoother asset loading, e.g. VS Code Live Server or:

   ```bash
   npx serve .
   ```

3. Scroll through the site and enjoy the animations.
